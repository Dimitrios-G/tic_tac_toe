# Tic Tac Toe Game

This is a web-based Tic Tac Toe game developed using HTML, CSS, and JavaScript. It offers both a two-player mode and a single-player mode against a computer AI with adjustable difficulty levels.

## Features

* **Player vs. Player Mode:** Play against a friend on the same device.
* **Player vs. Computer Mode:** Challenge yourself against an AI opponent.
* **Adjustable AI Difficulty:** Choose between Easy, Medium, and Hard difficulty settings for the computer opponent.
* **Symbol Selection:** Players can choose to play as 'X' or 'O'.
* **Scoreboard:** Tracks wins for Player X, Player O, and draws.
* **Game Status Display:** Provides real-time updates on whose turn it is or the game's outcome.
* **Responsive Design:** Enjoy the game seamlessly on various screen sizes (desktops, tablets, and mobile phones).
* **Audio Feedback:** Subtle sound effects for moves, wins, losses, and draws (can be toggled).
* **Confetti Animation:** A celebratory confetti animation for the winner.
* **Modal Pop-ups:** Informative pop-ups for game outcomes (win, lose, draw) with options for a new game or continuing.
* **Google Analytics Integration:** Basic event tracking for game actions (moves, wins, difficulty changes, etc.).

## Technologies Used

* **HTML5:** Structure of the game.
* **CSS3:** Styling and responsive design.
* **JavaScript (ES6+):** Game logic, AI, and interactivity.
* **Web Audio API:** For generating dynamic sound effects.
* **Google Analytics:** For usage tracking.

## How to Play

1.  **Open `Tic_Tac_Toe.html`** in your web browser.
2.  **Select Your Symbol:** Choose 'X' or 'O' at the top of the page.
3.  **Choose Opponent:** Select "Player" for a two-player game or "Computer" to play against the AI.
4.  **Select Difficulty (if playing against Computer):** If you chose "Computer" as your opponent, select "Easy", "Medium", or "Hard".
5.  **Start Playing:** Click on any empty cell on the board to make your move.
6.  **Game End:** The game ends when a player gets three of their symbols in a row (horizontally, vertically, or diagonally) or when all cells are filled, resulting in a draw.
7.  **New Game:**
    * After a game ends, a modal will appear with options for a "New Game" (resets scores and board) or "Continue" (resets only the board, keeping scores).
    * You can also click the "New Game" button below the board at any time to reset the current game.
8.  **Audio Toggle:** Use the speaker icon in the top right corner to enable or disable game sounds.
